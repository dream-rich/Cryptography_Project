OTP Based on AES and LCG

## Setup & Installation

```bash
pip install flask
```

```bash
pip install flask-mysql
```

## Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```

## Running The App

```bash
python main.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
